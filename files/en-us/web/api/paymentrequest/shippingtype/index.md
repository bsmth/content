The **`shippingType`** read-only property of the
{{domxref("PaymentRequest")}} interface returns one of `"shipping"`,
`"delivery"`, `"pickup"`, or `null` if one was not
provided by the constructor.

## Value

One of `"shipping"`, `"delivery"`, `"pickup"`, or
`null`.
